# Kanban

Simple kanban project demonstration using:
##### DDD -> Domain Driven Design
##### CQRS -> Command Query Responsibility Separation; and
##### ES -> Event Sourcing

## Features
* Board Creation
* Add Column to board
* Remove Column From Board
* Create WorkItem
* Add Workitem to column
* Remove Workitem From Column
* Progress WorkItem
